# A Parent's Guide to AI — Blog

This repository contains the built output for **[A Parent's Guide to AI](https://parentsguidetoai.ca)**, a blog by Yvan Boily.

The blog helps parents understand artificial intelligence and its impact on their children's lives — covering topics like deepfakes, privacy, AI in schools, mental health, scams, and practical hands-on guides for experiencing AI firsthand.
