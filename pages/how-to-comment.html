How to Comment or Report an Issue

Comments on this blog are powered by GitHub Discussions through a tool called Giscus. To leave a comment, you need a free GitHub account. This page explains how to get one and how to use it.

Why GitHub?

GitHub is a platform originally built for software developers, but it's free, widely trusted, and its account system is what allows comments here to be moderated, spam-free, and tied to a real identity. You don't need to understand software development to use it — you just need an account.

Pare of the reason I chose to use this approach to leaving comments is because I don't want to include a service that mantains or collects information or uses cookies or trackers. If anyone wants to come to this site and read the content they can, and if they want to engage further they can use Github, Reddit, or Facebook to engage!

Step 1 — Create a free GitHub account

  1. Go to github.com and click Sign up.
  2. Enter your email address and choose a username and password.
  3. Your username will be visible on any comments you post. Choose something you're comfortable with.
  4. Verify your email address when GitHub sends you a confirmation message.
  5. That's it. You don't need to fill in any profile information to use comments here.

GitHub accounts are free. You won't be asked for payment information.

Step 2 — Leave a comment on a post

  1. Scroll to the bottom of any blog post. You'll see a Comments section.
  2. Click Sign in with GitHub inside the comment box.
  3. GitHub will ask you to authorize the Giscus app — this is what connects GitHub to the comments here. Click Authorize giscus.
  4. You're now signed in. Type your comment and click Comment.

Your comment will appear immediately and notify me so I can respond.

A few guidelines:

  • Comments are moderated. Off-topic posts or anything disrespectful will be removed.
  • This blog is for parents navigating AI — questions, personal experiences, corrections, and suggestions are all welcome.
  • If you'd prefer not to create a GitHub account, you can email feedback directly through parentsguidetoai.ca.

Reporting a problem with the site

If you find a factual error, a broken link, or anything else that seems wrong, you can report it as a GitHub Issue:

  1. Go to github.com/ygjb/parents-guide-to-ai-blog/issues.
  2. Click New issue.
  3. Give your report a short title (e.g., "Broken link in Post 5" or "Factual error in AI scams post").
  4. In the description, include:
  5. Which post the problem is in
  6. What the error or problem is
  7. A link or quote if it helps
  8. Click Submit new issue.

I review issues regularly and will update the post if the report checks out.

Privacy note

When you sign in with GitHub to comment, Giscus can see your GitHub username and public profile. It does not access your repositories, private data, or anything beyond what's needed to post the comment. You can review Giscus's privacy policy if you'd like the details.
